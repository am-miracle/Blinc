WebGPU not available Blinc's web target requires WebGPU. Try Chrome / Edge 113+, or enable WebGPU in your browser.
On Safari Technology Preview and Firefox Nightly, WebGPU is behind a flag. wgpu also falls back to WebGL2 where WebGPU is unavailable, but some Blinc pipelines (storage buffers in the SDF aux buffer, particles) need WebGPU to be functional.
